Neural Network Indirect Encoding Method

This week I mainly focus on the research of the encoding method of Artificial Neural Network(ANN). Encoding plays a dominant role in neuralevolution, tradictional methods usually use direct encoding method to represent the structure of ANNs. However, direct encoding has it's limitation: it cannot handle todays deep neural network which has million of nodes and connections. In order to apply neuralevolution to higher dimension, indirect encoding method is introduced.

In his paper A Hypercube-Based Indirect Encoding for Evolving Large-Scale Neural Networks, K.O.Stanley proposes a method that use Compositional Pattern Producing Networks(CPPNs) to produce neural network pattern. CPPNs can represent connectivity patterns as functions of Cartesian space.

(a) The function f takes arguments x and y, which are coordinates in a two-dimensional space. When all the coordinates are drawn with an intensity corresponding to the output of f, the result is a spatial pattern, which can be viewed as a phenotype whose genotype is f.

(b) The CPPN is a graph that determines which functions are connected.
